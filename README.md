# Miniguia de Estudos – NotebookLM

## Contexto
O presente caderno temático foi desenvolvido com o objetivo de estudar [Assunto Financeiro Introdução: ex. Noções de Finanças Pessoais]. O foco do estudo foi compreender conceitos essenciais, boas práticas e aplicações no mundo real utilizando o NotebookLM como ferramenta de aprendizagem ativa.

Este projeto tem como finalidade treinar a Inteligência Artificial do NotebookLM para auxiliar no aprendizado, promovendo:

Curadoria de fontes confiáveis em formato de texto e PDF;
Organização das informações em resumos estruturados;
Aplicação de prompts estratégicos para explorar respostas detalhadas da IA;
Registro das dificuldades e ajustes feitos para otimizar a qualidade das respostas.

O caderno resultante permitirá criar um miniguia de estudo, incluindo resumos, glossário de termos e prompts reutilizáveis para revisões futuras.

## Objetivos
O presente caderno temático tem como objetivos principais:

Compreender os conceitos fundamentais relacionados ao assunto financeiro introdutório estudado.
Desenvolver a capacidade de aplicar perguntas estratégicas para explorar respostas da IA de forma eficiente.
Documentar de forma estruturada o conhecimento adquirido durante o estudo.
Criar um miniguia de estudo com resumos, glossário de conceitos e prompts reutilizáveis que possam ser utilizados em futuras revisões.

   ## Engenharia de Prompts e “Cicatrizes”

Durante o desenvolvimento do caderno temático no NotebookLM, foram criados prompts estratégicos para extrair informações úteis sobre o tema financeiro introdutório. Para cada prompt, registramos as respostas obtidas e os ajustes realizados para melhorar a qualidade das respostas.

**Prompt 1**  
Pergunta: "Explique os conceitos básicos de finanças pessoais."  
Variação testada: "Liste os principais conceitos de finanças para iniciantes em tópicos."  
Resposta obtida: O NotebookLM apresentou os conceitos de orçamento, poupança, investimentos iniciais e gestão de gastos.  
Dificuldades/Cicatrizes: A primeira resposta estava muito resumida, foi necessário ajustar o prompt para obter uma lista detalhada.

**Prompt 2**  
Pergunta: "Quais são as melhores práticas de proteção de dados financeiros pessoais?"  
Variação testada: "Explique de forma detalhada como aplicar a LGPD em finanças pessoais."  
Resposta obtida: Foram apresentadas recomendações de criptografia, senhas seguras, backup de informações e cuidados ao compartilhar dados.  
Dificuldades/Cicatrizes: Algumas respostas incluíam exemplos de TI corporativa, precisando adaptar para contexto pessoal.

**Prompt 3**  
Pergunta: "Crie um resumo prático sobre planejamento financeiro pessoal."  
Variação testada: "Resuma os principais passos para organizar finanças de forma simples e aplicável."  
Resposta obtida: O NotebookLM gerou um guia passo a passo para orçamento mensal, definição de metas e acompanhamento de despesas.  
Dificuldades/Cicatrizes: Ajustes foram feitos para simplificar a linguagem e focar em conceitos aplicáveis a iniciantes.

## Miniguia de Estudo

### Resumos Estruturados
- **Orçamento Pessoal:** Definir receitas e despesas mensais, controlar gastos e planejar poupança.  
- **Poupança e Investimentos:** Priorizar reserva de emergência, escolher investimentos de acordo com o perfil de risco.  
- **Segurança da Informação:** Utilizar senhas fortes, backups regulares, cuidado ao compartilhar dados.  
- **LGPD Aplicada:** Compreender os direitos sobre dados pessoais e aplicar boas práticas de proteção de informações.

### Glossário de Conceitos
| Termo | Definição |
|-------|-----------|
| Orçamento | Planejamento financeiro que organiza entradas e saídas de dinheiro. |
| Reserva de Emergência | Fundo financeiro destinado a despesas imprevistas. |
| Criptografia | Técnica de proteção de informações digitais para segurança dos dados. |
| LGPD | Lei Geral de Proteção de Dados Pessoais, regulamenta o uso de dados de indivíduos. |

### Prompts Reutilizáveis
- Prompt 1: "Explique conceitos básicos de finanças pessoais em tópicos."  
- Prompt 2: "Liste boas práticas de proteção de dados pessoais conforme LGPD."  
- Prompt 3: "Resuma planejamento financeiro de forma prática para iniciantes."  
- Prompt 4: "Crie um guia passo a passo para organizar orçamento mensal e investimentos iniciais."
